<h1>WinWifiDump</h1> <p><strong>A simple Windows tool that displays saved WiFi networks and their passwords.</strong></p> <hr/> <h2>Features</h2> <ul> <li><strong>Shows all WiFi profiles</strong> stored on the system.</li> <li><strong>Displays clear-text WiFi passwords</strong> (when available).</li> <li>Fast, lightweight, and easy to run.</li> </ul> <h2>Installation</h2> <ol> <li>Clone the repository:</li> </ol> <pre> <code>git clone https://github.com/vishnuuuvrdn/WinWifiDump.git</code> </pre> <ol start="2"> <li>Build the project or use the included executable (if available).</li> </ol> <h2>Usage</h2> <p><strong>Run Command Prompt as Administrator</strong> and execute:</p> <pre> <code>python WinWifiDump.py</code> </pre> <p>The tool will automatically list all WiFi profiles along with their passwords.</p> <h2>Project Structure</h2> <pre> <code> WinWifiDump/ ├─ WinWifiDump.exe ├─ source files ├─ README.md └─ LICENSE </code> </pre> <h2>Requirements</h2> <ul> <li>Windows operating system</li> <li><strong>Administrator permissions</strong> required to read WiFi passwords</li> </ul> <h2>Contributing</h2> <p>Pull requests and suggestions are welcome.</p> <h2>License</h2> <p>This project is licensed under the <strong>MIT License</strong>.</p>
